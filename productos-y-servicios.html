Productos y servicios

Servicios de Recaudación

1. Pago de cuentas o servicios

La red de recaudación de ZUMPAGO es un servicio que permite el pago de cuentas de productos y servicios utilizando los medios de pago disponibles, con la comodidad de tener todas las cuentas agrupadas en un solo lugar.

Características

  • Rápida integración con empresas de servicios.
  • Agrupación de cuentas a pagar por el cliente.
  • Estadísticas y gestión de pagos para clientes y comercios
  • Múltiples canales para pagos (Cajas propias y de comercios, Kioscos, Portal Web de pagos, App y otros).

Adicionalmente ZUMPAGO ha habilitado servicios adicionales para complementar y apoyar los procesos de recaudación de cuentas y servicios, como son:

a) Pago Fácil: Link que redirecciona al Cliente hacia la página web de ZUMPAGO, en donde el Cliente debe ingresar el identificador de la cuenta (ejemplo: Cédula, Código de Cliente u otro) para consultar su deuda y posteriormente pagar.

Características

  • Integración directa (sitio web, emailing, SMS, redes sociales, entre otros)
  • Simplifica el uso del canal web propio del comercio
  • Uso simple y experiencia de pago agradable para clientes finales ya que se accede al pago directamente
  • Requiere tener integrado previamente el servicio de recaudación

b) Botón de Redireccionamiento: Botón que redirecciona al Cliente hacia la página web de ZUMPAGO. Se invoca con el identificador de la Cuenta (ejemplo: RUT u otro), con lo cual el Cliente en el sitio de ZUMPAGO sólo debe ingresar su email (para el comprobante de pago) y presionar “Pagar”.

Características

  • Integración directa (sitio web, emailing, SMS, redes sociales, entre otros)
  • Aumenta y potencia el uso del canal web propio del comercio
  • Uso simple y experiencia de pago agradable para clientes finales ya que finaliza el pago con datos precargados, evitando el reingreso de datos
  • Requiere tener integrado previamente el servicio de recaudación

2. Recarga de Prepagos

A través de ZUMPAGO se puede realizar la recarga productos prepago de celulares, TV, internet u otros, recibiendo la confirmación de forma inmediata.

Características

  • Trabajamos con todas las compañías
  • Se puede recargar el monto que se desee.
  • Total seguridad y confiabilidad para las transacciones de recarga

5. Otros Servicios de Recaudación

Botón de Pago Multimedios

Servicio concentrador en línea de medios de pago que permite a los usuarios del comercio electrónico efectuar compras y pagos con cargo a diferentes medios de pago, tanto bancarios como de tiendas de retail u otros autorizadores.

Características

  • Queda integrado a la Tienda Virtual
  • El pago en internet es seguro, simple y al alcance de todos
  • Compatible con los diferentes sistemas operativos y browsers
  • Provee de un sistema de post venta para Usuarios, Comercios y Autorizadores
  • Incluye asesoría a los comercios para su implementación
Otros Servicios Transaccionales

ZUMPAGO dispone de una plataforma transaccional de pago flexible y que se ajusta a las necesidades y requerimientos de todo tipo de comercios, independiente de la industria, pudiendo aumentar sus servicios y mejorando sus procesos internos de forma rápida.

1. Portal de Pagos

Servicio ofrecido a las empresas de servicios para que sus clientes puedan pagar sus cuentas, cuotas, mensualidades, etc. Incluye la adaptación y configuración de un sitio web para el proceso de recaudación, con el look & feel de la empresa de servicios, el cual puede ser vinculado a su sitio web corporativo. Para el proceso de pago se habilita el Botón de Redireccionamiento para finalizar el proceso de pago.

2. Servicios a Pedido y a la Medida del Cliente

ZUMPAGO dispone de soluciones flexibles que permiten resolver necesidades específicas de clientes (custom solutions).
